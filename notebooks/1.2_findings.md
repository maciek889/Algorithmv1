# Stage 1.2 Findings - EDA + Horizon Calibration

## EDA Findings

- **Master dataset is usable for Stage 1.3:** `master_dataset.parquet` has 4779 rows, 11 expected columns, and spans 2007-01-03 to 2025-12-30.
- **Core missingness is clean after training start:** all core columns have zero NaNs from 2010-01-01 onward.
- **US100 returns are fat-tailed:** daily return kurtosis is about 8.0, so stress-window validation should matter more than average-period behavior.
- **Largest outliers look event-driven:** the biggest positive/negative days cluster around GFC, COVID, and 2025 tariff-shock dates rather than obvious data bugs.
- **Distribution shift is material:** 2019-2021 and 2022-2024 have higher volatility and volume than 2013-2018; later model training should test stronger recency weighting.
- **SPX is useful but partly redundant:** US100-SPX daily return correlation is about 0.93, so SPX context should be retained initially but tested for marginal value.
- **VIX is a strong inverse regime signal:** US100-VIX daily return correlation is about -0.70, supporting VIX-derived context features in Stage 1.3.
- **DXY is weaker context:** US100-DXY daily return correlation is about -0.14, so DXY should be treated as optional macro context.
- **Intraday range supports short horizons:** mean US100 intraday range is about 1.49%, with 90th percentile near 2.66%, consistent with ATR barriers resolving within weeks.
- **Calendar cadence is plausible:** completed years mostly match expected FOMC/CPI/NFP frequencies; partial boundary years explain CPI/NFP counts in 2007 and 2026.

## Horizon Calibration Findings

- **Calibrated horizon is 14 trading days:** this is the 75th percentile of hit-case `time_to_first_barrier` for 2010-2011 entries.
- **Simulation sample is sufficiently complete:** 504 entries were simulated, 503 hit a barrier, and only 0.1984% had no barrier hit within 60 trading days.
- **Hit-time distribution is concentrated:** mean hit time is 9.825 days, median is 8 days, p90 is 21 days, and max hit time is 47 days.
- **SL-first dominates in calibration:** 61.829% of resolved positions hit SL first versus 38.171% TP first, reflecting asymmetric barriers and the 2010-2011 regime.

## Conventions Used

- **Same-day TP/SL convention:** if both barriers are touched on the same future day, calibration records TP first. This is intentionally documented for calibration only and should be revisited for backtest realism in Stage 1.7.
