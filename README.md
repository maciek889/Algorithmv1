# US100 ML Trading System — Data Pipeline (Stage 1.1)

Foundation data pipeline for an ML-based trading system on the NASDAQ 100 index.
This stage produces a single master parquet of OHLCV + market context, plus
release-date calendars for macro events. Features, labels, and the model itself
arrive in later stages.

Setup and usage instructions are finalized at the end of Stage 1.1 (Phase 8).
